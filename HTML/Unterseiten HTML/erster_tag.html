Mittels des Tags <sub> kann man Text tief stellen.
Mittels des Tags <sup> kann man Text hoch stellen.
Mittels dem Attribut "style" kann man dem Text unter anderem Farbe oder Stil oder Größe geben.
Um ein einzelnes Wort im Paragraph zu stylen kann man das Tag <span> oder <mark> (für Screenreader) nutzen.

Für einen horizontalen Break nutzt man <hr> nutzen. 

            Der Tag <pre> stellt den Text formatiert da wie im SourceCode.

            Mit allen       Umbrüchen und Zeilenabständen.

Tabellen

Um einen Tabelle darzustellen nutzt man den Tag <table> in kombi mit <tr> für die jeweilige Spalte, <th> für die Header Zeile und <td> für die restlichen Zeilen.

Reihe 1 Reihe 2 Reihe 3
Wert 1 Wert 2 Wert 3 
            Für einen besseren Syntax schachtelt man das am besten in thead (Table-header) bzw. tbody (Table-body). Der Syntax würde dann so aussehen:
            

            <table>
                <thead>
                    <tr>
                        <th>1.Header</th>
                        <th>2.Header</th>
                        <th>3.Header</th>
                    </tr>
                </thead>
                <tbody>
                    <tr>
                        <td>1.Zeile</td>
                        <td>2.Zeile</td>
                        <td>3.Zeile</td>
                    </tr>
                </tbody>
            </table>
